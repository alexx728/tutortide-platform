# tutortide-platform
TutorTide — єдине середовище для організації онлайн-навчання. Допомагає учням ефективно знаходити викладачів, а репетиторам — керувати розкладом.

## Проблема
Учні стикаються з тривалим та хаотичним пошуком перевірених викладачів для онлайн-занять, а також зі складністю узгодження розкладів у різних месенджерах.

## Пітч
TutorTide — це платформа, яка вирішує проблему довгого пошуку репетиторів. Учні отримують зручне середовище для швидкого підбору викладача та проведення занять, а репетитори — інструмент для керування розкладом і залучення нових клієнтів.

## Ціль першої ітерації
Створити робочу гіпотезу з чітко сформульованою проблемою та розробити базовий функціонал пошуку викладачів, бронювання слотів і проведення перших відеодзвінків.

## Ядро цінності (MVP)
Визначається один ключовий сценарій, без якого цінність не реалізується: можливість зафіксувати запис, побачити його в списку та здійснити дзвінок. 
> 📄 *Детальний користувацький сценарій та критерії приймання знаходяться у файлі [`docs/MVP.md`](docs/MVP.md)*.

## Карта стейкхолдерів
Учні та студенти сприйматимуть цінність від швидкості пошуку. Репетитори оцінюватимуть досягнення цілей через зручність платформи. Засновники проєкту впливатимуть на реалізацію обмеженнями часу та ресурсів. Усвідомлене управління їхніми очікуваннями формує реалістичний план.

## Ознаки успіху
Це вимірювані умови, за яких вважається, що цілі досягнуто. Для першого етапу це:
1. Здатність виконати головний сценарій бронювання без зайвих кроків.
2. Стабільність простої операції протягом тижня використання.
> 📄 *Розширений план перевірки гіпотез та робота з невизначеностями описані у файлі [`docs/definition-of-success.md`](docs/definition-of-success.md)*.
